## 🎯 О проекте

Проект демонстрирует возможность автоматического определения спелости томатов с использованием глубокого обучения. Система классифицирует помидоры по степени зрелости для применения в сельском хозяйстве, автоматизации сбора урожая и контроле качества.

### Цели проекта

- Демонстрация применения YOLOv7 для сельскохозяйственных задач
- Определение степени спелости томатов на изображениях
- Показательный проект для обучения и презентации технологий

## 📊 Набор данных

### Описание

- **Тип данных**: Кастомный датасет, собранный из различных источников
- **Разметка**: Проводилась **вручную** для обеспечения высокого качества
- **Классы**: Томаты разной степени спелости (незрелые, частично спелые, спелые)
- **Формат**: YOLO формат (txt файлы с координатами bounding box)

### Характеристика датасета

| Параметр | Значение |
|----------|----------|
| Источник | Различные источники (кастомный сбор) |
| Разметка | Ручная разметка |
| Формат | YOLO |
| Классы | 3+ (незрелый, средний, спелый) |

## 🤖 Модель

### Архитектура

- **Нейросеть**: YOLOv7 (You Only Look Once v7)
- **Время обучения**: ~48 часов
- **Стратегия**: Выбор лучших весов (best weights)
- **Фреймворк**: PyTorch

### Обучение

**Параметры обучения:**
- Длительность: 48 часов
- Подбор лучших весов: включён
- Оптимизация: автоматический выбор лучших checkpoint'ов
